Como posso usar múltiplas condições com 'or'?

Você pode usar or para combinar condições onde pelo menos uma delas precisa ser verdadeira. Por exemplo: 

if tempo_ensolarado == "sim" or fim_de_semana == True:
    print("Vamos ao parque!")
Isso permite que a saída seja positiva se qualquer uma das condições for verdadeira.

Como posso usar múltiplas condições com 'and' e 'or' juntos?

Você pode combinar and e or para criar condições mais complexas. Por exemplo: 

if (tempo_ensolarado == "sim" or fim_de_semana == True) and energia_disponivel:
    print("Atividade Confirmada")
else:
    print("Atividade Cancelada")
Isso garante que a atividade só será confirmada se pelo menos uma das condições iniciais for verdadeira e se houver energia disponível.

Mas como garantir que as condições estão sendo verificadas na ordem correta?

Assim como nas operações aritméticas, você deve usar parênteses para garantir que as condições sejam avaliadas na ordem correta: 

if (tempo_ensolarado == "sim" or fim_de_semana == True) and energia_disponivel:
    print("Atividade Confirmada")
else:
    print("Atividade Cancelada")
Isso ajuda a evitar confusões com a precedência dos operadores lógicos.
